# Bankomat-OOP
 Bankomat med OOP 



 Instruktioner
## OOP Bankomat
Gör ett program enligt god Object orienterad programmerings standard


1. Skapa en klass som innehåller:
	- Kontonr
	- Namn
	- Räntesats
	- Saldo
	- MaxKredit


2. Det ska inte gå att ta ut mer än kreditgränsen på kontot

3. Det ska gå att skapa ett valfritt antal konton dynamiskt.


4. Ha menyvalen:

	- Gör en insättning på ett konto
	- Gör ett uttag på ett konto
	- Visa saldot på ett konto
	- Skriv ut en lista på alla kontonr och saldon


### VG

Allt för G 

Möjlighet att genom ett menyval göra utdelning av ränta till alla konton

Möjlighet att sortera utifrån Saldo och räntan både i stigande och fallande ordning
